IMTOC Image to CFG

Transforms an image converted to text with ImageMagick into an executable format for Sauerbraten


Data Reader

paste here the cubescript that will handle the color data (or use the default)

the conversion may take a while with large files

Guide:

First of all you need to choose one of the online tools to convert your image to a txt file (they all use ImageMagick to convert).

After downloading the txt file, you can import or paste it into the left text field (If you paste, the image name will not be saved).

The text field on the right is where the result will be stored after you click Convert, you can download the cfg (result) file by clicking on the last button.

What's Data Reader?

This field is dedicated to the CubeScript that is responsible for transforming the hexadecimal values ​​into cubes ingame.

The default is a simple data reader, not perfect but works fine, I hope someone more skilled make a better version of it.

It is added to the end of the output, after converting.

if you want to make your own (not necessary), the important variables are:

$imtoc_imagename

$imtoc_imagew

$imtoc_imageh

$imtoc_colordata

Ingame Guide:

After putting the cfg file in the sauerbraten folder and running it ingame...

Assuming you're using the default data reader...

/imtoc-reset will reset the 'pixel id' value to 0, you must do this every time you start a new image.

/imtoc-undo this subtracts the width value from the 'pixel id', then you can re-draw this line.

/imtoc imagename 0/1 this is the command that will actually draw the pixel/cube, the name of the image should be the cfg name you downloaded earlier. 0/1 disable or enable alpha.
